# Contributing

This project does not accept external code contributions - pull requests
will be closed unread. If you want to run a modified version of this addon
(or the companion scanner it talks to), fork it and self-host your own copy
instead.

This repo is licensed under [AGPLv3](LICENSE): you're free to fork, modify,
and distribute your own version, including publicly, as long as you make
your source (with modifications) available to your users.

Bug reports are still welcome via Issues.
